# Compactar o sistema (CompactOS)

Reduz uso de disco compactando binários do Windows com CompactOS.

## Verificar estado atual
```cmd
compact.exe /compactOS:query
```

## Ativar CompactOS
```cmd
compact.exe /compactOS:always
```

## Desativar CompactOS (se necessário)
```cmd
compact.exe /compactOS:never
```
